# VK Vote Audit

Инструмент аудита голосований ВКонтакте. Выявляет накрутку через анализ профилей проголосовавших.

## Что делает

1. **collect.py** — собирает данные через VK API:
   - Информация об опросе (варианты, голоса)
   - Списки проголосовавших (для неанонимных опросов)
   - Профили всех голосовавших (город, активность, фото, статус)
   - Список участников группы

2. **report.py** — генерирует HTML-отчёт с:
   - Оригинальные vs скорректированные результаты
   - Уровень риска накрутки по вариантам
   - География голосования
   - Полные списки подозрительных аккаунтов
   - Деактивированные/забаненные аккаунты
   - Итоговый вердикт с методологией

## Требования

- Python 3.10+ (stdlib only, без внешних зависимостей)
- VK API токен с правами `groups`, `stats`, `wall`

## Установка и запуск

```bash
git clone <repo-url>
cd vk-vote-audit

# 1. Настроить переменные
cp .env.example .env
# Отредактировать .env — указать VK_TOKEN и параметры опроса

# 2. Собрать данные
python collect.py

# 3. Сгенерировать отчёт
python report.py

# 4. Открыть отчёт
# Windows:
start output/report.html
# Linux/Mac:
open output/report.html
```

## Получение VK API токена

Самый простой способ — через OAuth Implicit Flow:

```
https://oauth.vk.com/authorize?client_id=2685278&scope=groups,stats,wall&redirect_uri=https://oauth.vk.com/blank.html&display=page&response_type=token&v=5.199
```

1. Откройте ссылку в браузере
2. Нажмите «Разрешить»
3. Скопируйте `access_token` из адресной строки
4. Вставьте в `.env`

Токен действует 24 часа.

## Конфигурация (.env)

| Переменная | Описание | Пример |
|---|---|---|
| `VK_TOKEN` | OAuth-токен VK API | `vk1.a.XXXX...` |
| `VK_WALL_POST` | Пост с опросом (ownerID_postID) | `-167291581_464` |
| `VK_GROUP_ID` | ID группы (без минуса) | `167291581` |
| `VK_TARGET_CITY` | Целевой город для гео-анализа | `киров` |

## Методология скоринга

Каждому проголосовавшему присваивается suspicion score:

| Критерий | Баллы |
|---|---|
| Деактивирован/забанен | +3 |
| Неактивен >180 дней | +2 |
| Нет аватарки | +1 |
| Не в группе | +1 |
| Город не указан | +1 |
| Город не совпадает с целевым | +1 |
| Закрытый профиль + нет фото | +1 |
| Нет данных об активности | +1 |

**Классификация:** 5+ = бот, 4 = подозрительный, 2-3 = слабые признаки, 0-1 = чистый.

Скорректированные результаты = оригинал минус аккаунты с score ≥ 5.
